##  Features

**Authentication & Authorization**: 
- User Registration & Login (JWT-based)
- Role-based Access Control (Admin, Customer)
- Secure Password Hashing (bcrypt)

**Vehicle Management**:
- Add, View, Update, Delete Vehicles
- Vehicle Availability Tracking

**Booking System**:
- Create, View, Update, Cancel Bookings
- Automatic Price Calculation (Daily Rate Ã— Days)

**User Management**: 
- User Deletion with Active Booking Check

**Admin Features**
- View All Bookings & Users
-  Mark Bookings as Returned
- Manage Vehicles & Users

##  Technology Stack


1. **Node.js with Express.js** - Server framework
2. **TypeScript** - Type safety & better development
3. **PostgreSQL (Neon DB)** - Cloud database
4. **JWT** - Authentication & authorization
5. **bcrypt** - Password hashing

##  Quick Start

### Prerequisites
- Node.js (version 18 or higher)
- PostgreSQL database (Neon DB recommended)
- Git
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ayathree/level2-assignment2.git
   cd level2-assignment2
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   http://localhost:5000
   ```
